# UI Components

Basic reusable UI components for the Playo frontend.

## Components to be created:

- **Button.tsx** - RPG-styled button component
- **Card.tsx** - RPG card wrapper component
- **Input.tsx** - Form input component
- **Modal.tsx** - Modal dialog component
- **Badge.tsx** - Badge/tag component for stats and labels
- **XPBar.tsx** - Experience points progress bar
- **StatBadge.tsx** - Stat display component (STR, WIS, etc.)
- **Toast.tsx** - Notification toast component
- **Spinner.tsx** - Loading spinner component
- **Tabs.tsx** - Tab navigation component

These will be implemented in the next steps.
